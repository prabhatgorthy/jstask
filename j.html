Double click Property


This method is a shortcut for .on( "dblclick", handler ) in the first two variations, and .trigger( "dblclick" ) in the third. The dblclick event is sent to an element when the element is double-clicked. Any HTML element can receive this event. We can do desired manipulations on the selected HTML element using the jQuery double click event handler.
